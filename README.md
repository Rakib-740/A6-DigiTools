# 🧰 DigiTools

A modern React-based platform where users can explore premium digital tools, add them to a cart, and manage their selections with a smooth and interactive experience.

---

## ⚙️ Features

- 🧩 Explore various premium digital tools
- 🛒 Add tools to cart
- ❌ Remove tools from cart
- 🔢 Real-time cart count in navbar
- 💰 Dynamic total price calculation
- 🔄 Toggle between Products & Cart view
- 💎 Three pricing plans — Starter, Pro, Enterprise
- ⚡ Toast notifications on add to cart, remove, and checkout using **React Toastify**

---

## 🧠 Tech Stack

- **React.js** – Frontend library
- **JavaScript (ES6+)** – Application logic
- **Tailwind CSS** – Styling
- **DaisyUI** – UI Components
- **React Toastify** – Notifications
- **Vite** – Fast development build tool
- **Vercel** – Deployment
